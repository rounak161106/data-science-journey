## Notes:
- Mean imputation works well with normally distributed data.
- If we have outliers and the distribution is right or left skewed, we can use median imputation.
- Mode imputation works well with categorical data.

### Handling imbalanced dataset
- Imbalanced data means that the no. of samples in one class is significantly higher than the other which can lead to biased ml models so we need to handle them. We can handle them by using techniques like oversampling (up-sampling), undersampling (down-sampling), and synthetic data generation (SMOTE).
